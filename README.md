# BG-PRODUCT-ANALYSIS-SALES
There is uneven sales performance and competition between chains, cities, territories. We are to find the products that drive revenue across these segments.

**PROBLEM STATEMENT** 
There is uneven sales performance and competition between chains, cities, territories. We are to find the products that drive revenue across these segments.

**DATA CLEANING** 
The following steps were taken to prepare, clean and transform the data;
•	We removed duplicates from the employee id column in the facts table 
•	Assigned the right data types to each of the columns 

KEY INSIGHTS 
•	Sales performance varies between territories  
•	Only a small number of products contribute to revenue 
•	There is strong competition that reduces sales impact in cities 
•	Some high performing territories contribute more value than the others 
•	
TOOLS AND TECHNIQUES USED
Power BI and Excel were used for analysis and dashboards. Key techniques include data cleaning, trend analysis, comparison of products sales performance across chains, territories in revenue 

RECOMMENDATIONS
•	Grow sales in average-performing areas by investing more in territories that are doing okay but still have room to grow, instead of focusing only on the top regions
•	Use sales efforts more effectively by moving effort to support areas that generate better returns and rewarding strong performance 
•	Improve partnership performance by working more closely with strong chains that are not doing well
•	Sell a wider range of products by promoting them through ads and campaigns and not focusing only on a few  products to generate revenue
•	Focus on products that clearly stand out from competitors and promote them more
